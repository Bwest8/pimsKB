# Appendix AR – Values to use in the Student Local Assessment Subtest Template

The tables below detail the values to use in the when reporting the results of local assessments in the Student Local Assessment Subtest Template

:::note

-   Where Test Description = LOCAL, the data is used for the local assessment option for the Early Indicators or Success measure in the Future Read PA Index.
-   Where Test Description is other than LOCAL, the data is used for reporting and analytics.
:::

## For reporting Local Assessment for Early Indicators of Success

| Test Description | Subtest Area | Subtest Version ID | Score Type 1 | Standard Achieved 1 | Student Met Standard                                                                              |
|------------------|--------------|--------------------|--------------|---------------------|---------------------------------------------------------------------------------------------------|
| `LOCAL`            | `MATH`         | `1`                  | `STD`          | `MOTS`                | `Y` if the student met the on-track standard or `N` if the student did not meet the on-track standard |
| `LOCAL`            | `READING`      | `1`                  | `STD`          | `MOTS`                | `Y` if the student met the on-track standard or `N` if the student did not meet the on-track standard |
